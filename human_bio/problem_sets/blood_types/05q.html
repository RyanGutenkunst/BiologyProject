Answer to First Message by Professor Hallick:
Blood type analysis can occasionally rule out a potential father (i.e. if the mother is type A and the child is type O, the father could not be type AB, but could be type A, B, or O), but is not a method that can be used to establish paternity.

If the paternity of your grandson needs to be established for legal reasons, such as establishing responsibility for child support, a DNA test would be the accepted method. There are commercial laboratories that will do the test for a fee (several hundred dollars). Blood samples would be required from the mother, child, and any alleged fathers. You might want to visit the Blackett Family Activity to learn more about DNA testing.

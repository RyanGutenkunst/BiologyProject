Nucleic Acids and the Genetic Material Problem Set 1

Problem 13: mRNA polypeptide code

Which mRNA codes for the following polypeptide?

met arg ser leu glu

The genetic code

 The correct codons, starting with 5'-AUG for methionine can be read off the table. One of the stop codons must be present as well. Remember that the codon is the same whether written with 5' on the left or reversed with 5'- on the right. For example, 5'GUA = 3'AUG

Regulation of Carbohydrate Metabolism Problem Set

Question 8: Insulin action in (skeletal) muscle

Insulin regulates all of the following enzymes in liver. Which of these enzymes are also regulated by insulin in muscle?

  1. Glycogen synthase

  2. Glycogen phosphorylase

    Recall that glucokinase and PFK-2 are liver specialties. Since PFK-1 is regulated indirectly via PFK-2, that also eliminates PFK-1. The only ones left are two enzymes controlling glycogen metabolism. Muscle does not contain glucokinase but instead contains hexokinase (which is not regulated by insulin).


  3. Phosphofructokinase-1 (PFK-1)
  4. Phosphofructokinase-2 (PFK-2)
  5. Glucokinase

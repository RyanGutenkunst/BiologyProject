Features of the equation for the dark reaction
RUBISCO is the CO2-fixing enzyme for this process. See the tutorial for problem #1 of problem set 2 of Photosynthesis.

The reactions occur in the chloroplast stroma. See the tutorial for problem #1 of problem set 1 of photosynthesis. and the tutorial for problem #2 of problem set 2 of photosynthesis.
